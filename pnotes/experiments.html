1.14 Notes

  1. To generate the text file for training, use $PROJECT_DIR/src/deeplab-v2/exper/scripts/generate_image_set.py
  2. augmentImages.m now generates a text file which contains all augmented images. I think you should generate the augmentations for all categories (and generate corresponding text files). Then you can combine the text files for the intended purpose.
  3. $PROJECT_DIR/data/VOCdevkit/VOC2012/Red_Annotations_Part contains reduced (i.e. clubbed parts) annotations for all categories. Try to not to remove/modify the contents of this directory unless you discover an error.
