3.1 TODO Learning Deep Representation with Large Scale Attributes

Links:

  • ICCV Paper

3.1.1 Ideas

To estimate pose, use:

  • Use the aspect ratio of the width and height
  • Attributes data : Use this to get object attributes (pose, viewpoint, keypoints specific attributes, inter-class attributes).

3.1.2 Contributions of the Paper

  1. They describe an attribute dataset on over 180k images, 499 classes and attributes including rotation, viewpoint, part occlusion, part existence, object part location.
  2. Improvement of object detection using earlier methods with the attributes.

    They trained the network to predict attribute mixtures instead of single attributes.

  3. Hierarchical clustering based on attributes to understand importance of attributes in capturing variation.

3.3 TODO DeepLab: Semantic Image Segmentation with Deep Convolutional Nets, Atrous Convolution, and Fully Connected CRFs

3.3.1 Challenges in Semantic Segmentation

  1. Reduced feature resolution
  2. Existence of objects at multiple scales

3.3.2 Contributions

  1. Uses atrous convolution which
    • enlarges the field of view without increasing number of parameters or computation

3.3.3 Method

A deep convolutional neural network (VGG-16 1 or ResNet-101 2 in this work) trained in the task of image classification is re-purposed to the task of semantic segmentation by (1) transforming all the fully connected layers to convolutional layers (i.e ., fully convolutional network 3) and (2) increasing feature resolution through atrous convolutional layers, allowing us to compute feature responses every 8 pixels instead of every 32 pixels in the original network. We then employ bi-linear interpolation to upsample by a factor of 8 the score map to reach the original image resolution, yielding the input to a fully-connected CRF 4 that refines the segmentation results.
